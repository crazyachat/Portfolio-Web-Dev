# 💼 Lakshya Singh - Personal Portfolio Website

A modern, elegant, and fully responsive personal portfolio website built using **HTML5**, **CSS3**, and **Vanilla JavaScript**.

This portfolio showcases my technical skills, education, certifications, projects, and professional contact information while maintaining a clean, premium UI inspired by modern web design principles.
